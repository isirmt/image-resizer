# 画像の縮小化アプリケーション

![thumbnail](https://img-sm.isirmt.com/ogp.png)

画像を簡易的に縮小化できます．

- 👍️: ファイルサイズを参照しつつ可能
- 👍️: クリップボード操作に対応

## 仕様

キャンバスのサイズごと変更することで拡大縮小を行います．マウスが離れた際に拡大縮小処理を挟んでいます．

同時に，出力サイズを得ることで特定のファイルサイズまで落とすことも可能です．
